# My Twitter

I made a copy of twitter application in React Native.

## Installation

- npm install
- pods install
  try it and enjoy

## Development

|         | Debug                         |
| ------- | ----------------------------- |
| android | $npx react-native run-android |
| ios     | $npx react-native run-ios     |
